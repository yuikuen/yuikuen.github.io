CentOS7部署_MySQL5.7(编译安装)
2022-04-07|lanmp
Word count: 896|Reading time: 4 min

前期准备

1）删除旧版本的 MySQL 及相关配置文件

2）安装依赖并下载程序包

安装 MySQL

1）添加 mysql 组和 mysql 用户

2） 创建配置文件

3）修改当前目录拥有者，并初始化 mysqld

4）设置开机自启

5）环境变量配置

将 mysql 的 bin 目录加入 PATH 环境变量，编辑 /etc/profile 文件

配置 MySQL

1）测试登录并修改密码

2）开启远程登录

  • 确认关闭或开启防火墙 mysql-3306 端口的外部访问
  • 使用 root 需要赋予远程权限
