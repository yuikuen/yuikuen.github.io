git操作_更新fork的项目为最新版本
2022-04-07|git
Word count: 437|Reading time: 2 min

在 github 上 fork 了一个项目之后，如何使自己 fork 的项目和原先作者的项目分支保持同步呢，下面使用 k8s 的项目做示范，示范如何使 fork 的项目更新保持同步。

1）首先 Clone 项目到本地

2）查看当前远程版本库地址

3）添加原项目 git 地址到本地版本库

4）检查版本是否添加成功

5）原项目更新内容同步到本地

6）查看本地分支

7）同步更新内容到本地对应分支

8）提交更新内容到 fork 地址

最后自行到 Github 仓库查看是否有 update 成功；
