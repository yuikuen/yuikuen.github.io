Helm部署_Ingress-Nginx(v0.49.0)
2022-04-07|helm
Word count: 987|Reading time: 5 min

环境说明

版本信息

  • Kubernetes：v1.20.9
  • Helm：v3.5.4
  • Ingress-Nginx：v3.29.0 & 0.45.0

Ingress-Nginx 官方推荐使用 Helm 进行安装，Helm 安装请自行参考 Kubernetes 部署-Helm

镜像文件：操作过程中，需要提前使用 Github & 阿里云 进行同步镜像

部署安装

1）添加 ingress-nginx 源仓库，并下载配置文件

2）修改 image 地址、部署模式、添加标签等操作

3）修改 image，因国外镜像，国内无法下载，直接执行 helm install 是无法成功的，需要提前做好镜像；

4）采用宿主机的网络，开启 hostnetwork 模式，并使用 Daemonset 指定 Node 作前端(因使用 deployment 创建 PO 会偏移)

5）为 Node 添加标签并开始安装

验证功能

创建简单的 Nginx & svc 进行测试

  • 方法二

扩缩方式

因为采用的是 DaemonSet 的方式部署，扩缩方式同理，直接为 Node 新增&删除 label 并且修改 hosts 文件或域名解析即可；
