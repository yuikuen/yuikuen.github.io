014_Jenkins-Githook自动触发构建
2022-04-07|learn
Word count: 207|Reading time: 1 min

轮询 SCM 可以实现 Gitlab 代码更新，项目自动构建，但定时扫描整个项目的代码，无疑是增加系统的压力。而更好的方案就是利用 Gitlab 的 webhook 实现代码 push 到仓库，立即触发项目自动构建

安装Gitlab hook 插件，设置自动构建

Gitlab 配置 Webhook

1）使用 root 账户登录到后台，点击 Admin Area > Settngs > Network

勾选”Allow requests to the local network from web hooks and services”

2）在项目添加 webhook

点击项目 > Settings > webhooks

注意事项：需要提前到 Jenkins 配置，否则会报错

修改 Jenkins Gitlab 配置，再次测试

通过修改代码并提交 push，即自动发起构建任务
