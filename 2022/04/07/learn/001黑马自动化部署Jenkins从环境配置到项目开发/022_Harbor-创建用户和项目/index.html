创建用户和项目

1）创建项目

Harbor 的项目分为公开和私有的：

  • 公开项目：所有用户都可以访问，通常存放公共的镜像，默认有 library 公共项目
  • 私有项目：只有授权用户才可以访问，通常存放项目本身的镜像

2）创建用户

创建用户 spring/spring123

3）给私有项目分配用户

进入 spring 项目 > 成员

角色 权限说明
访客 对于指定项目拥有只读权限
开发人员 对于指定项目拥有读写权限
维护人员 对于指定项目拥有读写权限，创建 webhooks
项目管理员 除了读写权限，同时拥有用户管理/镜像扫描等管理权限

镜像上传下载

把镜像上传到 Harbor

1）给镜像打上标签

2）推送镜像

如报错，是因为 Docker 没把 Harbor 加入信任列表中

3）把 Harbor 地址加入到 Docker 信任列表

4）重启 Docker，再次推送命令，会提示权限不足，需要登录 Harbor，再推送镜像

从 Harbor 下载镜像

1）安装 Docker，并正常启动 Docker

2）将 Harbor 地址加入 daemon.json 信任列表，重启服务

3）先登录，再从 Harbor 下载镜像
