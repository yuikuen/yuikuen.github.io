环境部署：

  • 系统环境：ESXi-6.7.0 + CentOS-7-x86_64-Minimal-2009
  • 安装版本：docker-ce-19.03.15 + docker-compose-linux-x86_64(v2.2.2) + harbor v2.4.1

开放对应服务，或直接关闭防火墙

安装 Docker

Docker 要求系统内核版本高于3.10，安装前请确认系统内核版本是否支持

安装需要的软件包，yum-util 提供 yum-config-manager 功能，另外两个 是devicemapper 驱动依赖的

由于国内访问不到 docker 官方镜像的缘故，可以通过 aliyun 的源来完成

安装 docker，默认最新版(docker-ce:社区版 ee:企业版)

参考：安装特定docker版本(先列出列出可用版本)–可选择性安装

所有节点设置防火墙规则，并让生效(关闭防火墙可略过)

镜像加速器：阿里云加速器，daocloud 加速器，中科大加速器
Docker 中国官方镜像加速：https://registry.docker-cn.com

安装 Docker-compose

Docker Compose 是 docker 提供的一个命令行工具，用来定义和运行由多个容器组成的应用，使用 compose，可以通过 YAML 文件声明式的定义应用程序的各个服务，并由单个命令完成应用的创建和启动

安装 Harbor

本文采用 http 和离线安装方式，如需使用 https + 域名方式，请参考 Harbor 官方文档

1）修改配置文件，若是采用 http 方式搭建，可忽视 https 的配置部分，将其注释掉

执行预备脚本 ./prepare，待测试完出现 Successfully called func: create_root_cert 表示可正常部署

2）执行安装脚本(--with-chartmuseum 是安装chart仓库，不使用helm可不添加该参数)
